# -
پروژه که برای مدیریت ورود و خروج دانشجویان، تخصیص میز و کامپیوتر، ثبت گزارش خرابی سیستم‌ها و مدیریت منابع سخت‌افزاری طراحی می‌شود. این سیستم به ادمین کمک می‌کند تا بدون نیاز به بررسی دستی یا تجهیزات سخت‌افزاری پیچیده، بتوانند وضعیت حضور دانشجویان و استفاده از تجهیزات را به‌صورت دیجیتال مدیریت کنند



نحوه اجرای پروژه:  

در ترمینال دیگری به مسیر "Faculty-website-management-system\backend>" در فایل main.py . دستور uvicorn main:app --reload را بزنید تا backend را اجرا کنید.

در ترمینال به مسیر "Faculty-website-management-system\view\frontend>" ترمینال را باز کنید و در ترمینال دستور npm run dev را بزنید تا fronted را اجرا کنید.

سپس لینک داده شده را در ترمینال باز کنید.


توجه: پروژه وابستگی های زیادی نیاز دارد و بدیلیل قطعی اینترنت امکان اینکه پروژه را داکر بیس بیاریم بالا وجود نداشت. به محض استیبل شدن اینترنت. پروژه آپدیت میشود.
پ.ن: پروژه همچنان اشکالات محدودی دارد و جای بهتر شدن را دارد. ولی به دلیل قطعی گسترده طولانی مدت اینترنت کشور. امکان به روز رسانی پروژه وجود ندارد.
