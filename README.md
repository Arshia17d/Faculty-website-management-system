# -
پروژه که برای مدیریت ورود و خروج دانشجویان، تخصیص میز و کامپیوتر، ثبت گزارش خرابی سیستم‌ها و مدیریت منابع سخت‌افزاری طراحی می‌شود. این سیستم به ادمین کمک می‌کند تا بدون نیاز به بررسی دستی یا تجهیزات سخت‌افزاری پیچیده، بتوانند وضعیت حضور دانشجویان و استفاده از تجهیزات را به‌صورت دیجیتال مدیریت کنند
